## REQUIREMENTS :

   * `storage 400MB`
   * `python`
   * `git`
   * `internet`

## INSTALLATION TERMUX :

     apt update && apt upgrade -y
     apt install git python -y
     git clone https://github.com/DilumBBandara/D-Zip.git
     cd D-Zip
     python D-Zip.py
